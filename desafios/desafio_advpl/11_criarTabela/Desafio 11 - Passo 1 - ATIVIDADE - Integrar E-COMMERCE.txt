Objetivo: Integração de E-COMMERCE utilizando arquivo TEXTO

Meta....: Importar os dados originais de uma planilha para o Protheus

Cenario:
========
Existem dois sistemas que devem conversar entre si (Loja virtual e Protheus)

>> Loja Virtual (Front End - Na visão do Cliente)
>> Protheus     (Back End  - ERP=Sistema que gerencia todos os departamentos)

Quando alguém na loja virtual preencher os dados do cliente, deve-se enviar os mesmo dados ao Protheus (SA1-Cliente) 

O cadastro deve ser "Higienizado ou tratado" 
Higienização = Retira caracteres especiais: [ ^~´#"'!@#$%¨&*&()_-+=[]{} ]
Tratamento   = Complementar informação faltantes
